# Unemployment Analysis with Python

This project is developed as part of the CodeAlpha Data Science Internship.

## Objective

To analyze unemployment trends in India using Python and data visualization techniques.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Dataset

The project uses the Unemployment in India dataset.

## Features

- Data Cleaning
- Exploratory Data Analysis
- Statistical Summary
- Region-wise Analysis
- Covid-19 Impact Analysis
- Data Visualization

## Project Outcome

The analysis helps understand unemployment trends and the impact of Covid-19 on employment conditions in India.
